# Example mined skills

Three real skills mined by skill-extractor from the author's own Claude Code
sessions, exactly as the tool installed them (provenance anonymized). They show
the range of what gets mined: a guardrail distilled from repeated corrections,
a workflow observed from successful sessions, and a cleanup rule learned from a
mistake.

Scores in each footer are the miner's confidence/utility blended with the
measured outcome of the source trace. Note the first one carries `trace
outcome meh`: outcome weighting is honest, and a guardrail expressed in an
imperfect session is still a valid guardrail.
